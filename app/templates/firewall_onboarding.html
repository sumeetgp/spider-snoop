SECURE YOUR LLM TRAFFIC

Integrate SpiderCob's AI Firewall in 2 minutes. Protect against PII leaks, block unauthorized models, and audit every prompt.

1

Change Base URL

Replace standard OpenAI endpoints with our secure proxy. It acts as a transparent middleman.

OpenAI Python SDK 
from openai import OpenAI

client = OpenAI(
    api_key="your_spidercob_api_token",  # Authenticate with SpiderCob
    base_url="https://{{ host_url }}/v1/proxy"  # Route via Firewall
)

completion = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello world!"}]
)
2

Verify Protection

Try sending a test prompt containing sensitive data (e.g., a credit card number or "SECRET_KEY"). The firewall should redact or block it.

Test with cURL 
curl https://{{ host_url }}/v1/proxy/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "model": "gpt-3.5-turbo",
    "messages": [
      {"role": "user", "content": "My secret password is..."}
    ]
  }'
